1. Напишите программу, которая выводит через document.write все числа от 1 до 100, с двумя исключениями.
    • Для чисел, нацело делящихся на 3, она должна выводить ‘Fizz’
    • Для чисел, делящихся на 5 (но не на 3) – ‘Buzz’
      • Когда сумеете – исправьте её так, чтобы она выводила «FizzBuzz» для всех чисел, которые делятся и на 3, и на 5.
  2. Пользователь вводит строку текста (неважно какого, могут быть числа, символы и пр.).
    • Выяснить, является ли введённая строка палиндромом
    • Палиндром - слева направо и справа налево читается одинаково.пример: 707, топот
  3. Дана строка, все буквы которые написаны в нижнем регистре перевести в верхний и наоброт
    • К примеру - 'JavaScript' ==> 'jAVAsCRIPT'
  4. Напишите функцию, которая сравнивает две строки и определяет являются ли они анаграммой.
      Введите два слова и программа скажет, являются ли они анаграммой? К примеру:
    • Введите первое слово: апельсин
    • Введите второе слово: спаниель
    • Результат: слова "апельсин" и "спаниель" являются анаграммой.
  5. У массивов есть метод reverse, меняющий порядок элементов в массиве на обратный.
      В качестве упражнения напишите функцию:
    • Которая получает массив как аргумент и выдаёт новый массив – с обратным порядком элементов.
    • Не использовать встроенный метод reverse
  6. Напишите функцию поиска самого большого числа во вложенном массиве внутри массива
    • Сформируйте новый массив из наибольших чисел
    • Отобразите результат с помощью document.write
  7. Напишите функцию генерации псевдослучайного цвета (используя форму записи шестнадцатиричного кода цвета)
    • К примеру: #DC143C
  8. Создайте собственную реализацию функции filter(). Используя функцию обратного вызова.
    • filter([1, 2, 3, 4], callback); callback: n => n < 3 // [1, 2]
